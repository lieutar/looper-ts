# dxml-cli

CLI interface for  [dxml](https://github.com/lieutar/ts--dsml--dxml) .

## SYNOPSIS

``` bash
$> dxml -s md -o output.md input.dxml
```
## OPTIONS

### `--output``-o`

Output file path.

### `--lang`

Target language for trimming/processing

### `--pp``-p`

Preprocessor.

### `--style``-s`

Specify output style
